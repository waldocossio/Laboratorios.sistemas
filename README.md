# Laboratorio 2 - Evolución del Flujograma Académico

## Descripción
Este proyecto es una aplicación de consola desarrollada en C# que calcula el promedio final de un estudiante incorporando un sistema de bonificación. El programa valida los datos de entrada de forma segura, aplica reglas de cálculo específicas y clasifica el rendimiento académico final.

## Funcionalidades principales
* **Validación de datos:** Asegura que el usuario ingrese valores numéricos correctos mediante `TryParse`.
* **Cálculo de bono:** Extrae los últimos dos dígitos del carnet de identidad y calcula el 7% de ese valor para sumarlo como bonificación.
* **Clasificación académica:** Evalúa el promedio final y asigna una categoría de rendimiento.

## Tecnologías utilizadas
* C#
* .NET (Console Application)
* Visual Studio Community
